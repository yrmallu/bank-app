# BANK APP

Rails Banking App for Deposit, withdraw and Transfer of Balance

## Prerequisites

1) `Ruby version > 2.7`
2) `Rails version > 6`
3) `MySQL / PostgreSQL`


## Installation

1) `git clone ...`
2) `cd bank-app`
3) `change database.yml as per your database`
4) `bundle install`
5) `rake db:setup`
6) `rails s`

## Test

`bundle exec rspec`
